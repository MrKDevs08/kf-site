Command Documentation

Below is a detailed reference of all available KeyFlow slash commands.

/generate
Admin
▼

Make a new license key. Only admins can use this to create keys and set how long they last.

Parameters

Parameter Type Required Description
role Role Yes The role that will be granted to the user who redeems this license.
time String Yes Duration for the role license (e.g. 1s, 1m, 1h, 1d).
amount 💎 Integer No Number of keys to generate. Premium Tier required for bulk generation.
key_format 💎 String No Custom design structure using 'X' for random chars. Premium Tier required.
/redeem
User
▼

Use a license key to get your premium roles. Just run the command, enter your key, and the bot will give you access.

Parameters

Parameter Type Required Description
key String Yes The license key to redeem. Starts with SVR- for server upgrades or standard code for roles.
/profile
User
▼

Check your account. See what licenses you have and when they expire.

No parameters required. Fetches and displays your active role licenses for this server.

/dashboard
Admin
▼

An admin menu to manage your server. See stats, check keys, add time to licenses, and take away access from a simple menu.

No parameters required. Launches the visual management board for server administrators.
