Command Documentation

Below is a brief explanation of all available KeyFlow slash commands.

/generate

Generate a new license key. This command is restricted to administrators and allows you to create custom keys, specify expiration times, and manage the access level of the generated license.

/redeem

Redeem an active license key to gain access to premium features or roles. Users can run this command, enter their valid key, and the bot will automatically grant the associated permissions.

/profile

View your active licenses, current subscriptions, and expiration dates. This command provides a personalized overview of your account status within the server.
